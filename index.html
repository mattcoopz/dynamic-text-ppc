A) Admin options - top

This plugin will add options to the Settings > General page. At the bottom of the page you will see the four options you can set.
Each one of these options is to set a fallback so you don't need to set a parameter each time you set a shortcode.


Parameter 1 is the parameter that the shortcode will look for within the URL.
Parameter 2 will also be searched for within the URL if Parameter 1 is not found and no parameters are set in the shortcode.
Default style is the style you can set to fallback to in case you have not set one in the shortcode.
Use page title is cover the chance of a URL parameter not being passed at all or not being set in either fallback or the shortcode.

B) CSS Files and Structure - top

I'm using only one CSS files in this plugin and it there to store only the default heading styling that will be called if nothing set in either the fallback option or the shortcode call.

.defaultHeading {
font-size: 42px;
color: #000;
}
